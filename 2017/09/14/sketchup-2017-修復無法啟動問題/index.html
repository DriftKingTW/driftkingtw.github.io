SketchUp 2017 修復無法啟動問題

今天打開SketchUp突然跳出這玩意…

試過重新安裝和網路上的各種方法也沒有效…就是開不了 弄了老半天簡直要發瘋
直到我傳送了錯誤報告, 跳出這個頁面有推薦的解決方法
OS:你怎麼不再錯誤頁面跟我說(#`Д´)ﾉ

按照上面的教學, 到設定程式那邊把Microsoft Visual C++ 2015相關的套件全數移除

然後把SketchUp卸載之後重新安裝一遍, 就成功解決問題囉(^u^)
